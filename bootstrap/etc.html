닫기 아이콘, 캐럿(드롭다운)

float

.pull-left: float: left;
.pull-right: float: right;

네비게이션 바에서 컴포넌트를 정렬하려면 .navbar-left나 .navbar-right 사용

.center-block: 콘텐츠 블록 중앙 정렬

show, hidden

.show: display: block;
.invisible: 요소의 가시성을 토글하기 위해 사용
.hidden: display: none;
.hide: 항상 스크린리더에 효과가 있지 않아 .hidden이나 .sr-only 사용
.sr-only, .sr-only-focusable스크린 리더를 제외한 모든 기기에서 요소 숨기기

.text-hide: 텍스트 콘텐츠를 배경이미지로 대체할 때 사용
