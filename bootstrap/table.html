테이블 예제 만들기
약간의 패딩과 수평 구분선이 있는 기본 스타일의 .table 클래스 tbody 안의 테이블 행에 얼룩말 같은 줄무늬를 추가하려면 .table-striped 클래스
줄무늬 행 테이블은 :nth-child로 꾸며지기 때문에 익스 8에서는 지원 X 		테이블의 모서리와 셀의 선을 그리기 위해 .table-bordered 클래스
tbody 안의 테이블 행에 hover 상태를 확인하려면 .table-hover 클래스 셀 패딩을 반으로 줄여 테이블은 작게 만드려면 .table-condensed 클래스
.active : 특정한 행이나 셀에 hover 했을 때와 같은 색을 입힙니다. .success : 성공적이거나 긍정적인 액션을 의미합니다. .info : 중립적인 정보의 변화나 액션을 의미합니다.
.warning : 주의가 필요한 경고를 의미합니다. .danger : 위험하거나 부정적일수 있는 액션을 의미합니다. tr에도 해당 클래스명을 사용할 수 있다.
